## Available Scripts

In the project directory, you can run:


>Default Create-React-App scripts:
>>  npm `start`
>>
>>  npm `test`
>>
>>  npm `run build`
>>
>>  npm `run eject`
>
> JSON server:
>> npm `run server`
>>
